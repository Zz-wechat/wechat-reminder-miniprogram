# 微信提醒工具 - WeChat Reminder Mini Program

## 🎯 项目简介
一个简单易用的微信小程序提醒工具，帮助用户设置和管理日常提醒。

## 📱 功能特性
- 创建单次/重复提醒
- 微信消息通知
- 简洁直观的界面
- 本地数据存储

## 🚀 快速开始
1. 下载微信开发者工具
2. 导入 `miniprogram/` 目录
3. 使用测试号开始开发

## 📁 项目结构
```
wechat-reminder-miniprogram/
├── miniprogram/     # 小程序源代码
├── web-preview/     # 网页预览版
├── docs/           # 开发文档
├── examples/       # 学习示例
└── README.md       # 项目说明
```

提取脚本 extract.js (上传后运行)
// 提取脚本 - 在浏览器控制台运行此脚本
// 步骤：
// 1. 打开此HTML文件
// 2. 按F12打开开发者工具
// 3. 复制此代码到Console
// 4. 按回车运行

function extractAllFiles() {
    console.log('开始提取所有文件...');
    
    // 获取所有文件内容
    const files = document.querySelectorAll('.file');
    const fileData = {};
    
    files.forEach(file => {
        const filename = file.querySelector('.filename').textContent;
        const content = file.querySelector('pre').textContent;
        fileData[filename] = content;
    });
    
    // 创建下载链接
    for (const [filename, content] of Object.entries(fileData)) {
        const blob = new Blob([content], { type: 'text/plain' });
        const url = URL.createObjectURL(blob);
        const a = document.createElement('a');
        a.href = url;
        a.download = filename.split(' ')[0];
        document.body.appendChild(a);
        a.click();
        document.body.removeChild(a);
        URL.revokeObjectURL(url);
        console.log('已提取:', filename);
    }
    
    console.log('✅ 所有文件提取完成！');
    console.log('📁 文件已下载到本地');
    console.log('🚀 请上传到GitHub仓库');
}

// 自动运行提取
setTimeout(extractAllFiles, 1000);
部署指南 deployment-guide.md
# GitHub部署指南

## 步骤1：上传文件到GitHub
1. 访问：https://github.com/Zz-wechat/wechat-reminder-miniprogram
2. 点击"Add file" → "Upload files"
3. 上传此HTML文件（wechat-reminder-all-in-one.html）
4. 点击"Commit changes"

## 步骤2：提取文件（可选）
1. 在本地打开上传的HTML文件
2. 按F12打开开发者工具
3. 在Console中运行提取脚本
4. 文件将自动下载

## 步骤3：配置GitHub Pages
1. 进入仓库 → Settings → Pages
2. Source: 选择"Deploy from a branch"
3. Branch: main, Folder: /web-preview
4. 点击Save

## 步骤4：访问预览
等待1-2分钟，访问：
https://zz-wechat.github.io/wechat-reminder-miniprogram/

## 备用方案：直接使用
如果不想提取，可以直接：
1. 复制web-preview部分的代码
2. 创建新文件 web-preview/index.html
3. 粘贴代码
4. 配置Pages
