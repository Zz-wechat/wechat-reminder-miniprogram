# 🚀 微信提醒小程序 - 从零开始开发指南

## 第一步：安装开发环境

### 1. 下载并安装软件
1. **微信开发者工具**：
   - 访问：https://developers.weixin.qq.com/miniprogram/dev/devtools/download.html
   - 下载适合你系统的版本（Windows/macOS）
   - 双击安装，完成后打开

2. **Visual Studio Code**（可选但推荐）：
   - 访问：https://code.visualstudio.com/
   - 下载安装，用于代码编辑

### 2. 创建第一个项目
1. 打开**微信开发者工具**
2. 点击"新建项目"
3. 填写项目信息：
   - **项目名称**：MyReminder
   - **目录**：选择`/root/.openclaw/workspace/projects/MyReminder`
   - **AppID**：点击"测试号"（先用测试号开发）
   - **开发模式**：小程序
   - **模板**：空白模板
4. 点击"新建"

## 第二步：理解项目结构

创建成功后，你会看到这样的文件结构：

```
MyReminder/
├── pages/              # 页面文件夹
│   └── index/         # 首页
│       ├── index.js
│       ├── index.json
│       ├── index.wxml
│       └── index.wxss
├── utils/             # 工具函数（后续添加）
├── app.js            # 小程序入口
├── app.json          # 全局配置
├── app.wxss          # 全局样式
└── project.config.json # 项目配置
```

## 第三步：运行第一个小程序

1. 在微信开发者工具中，点击左上角的"编译"按钮
2. 等待编译完成，右侧会显示小程序界面
3. 你应该能看到一个简单的首页，显示"微信提醒工具"

## 第四步：开始编码学习

### 学习路线图：

#### 第一课：理解WXML（类似HTML）
```html
<!-- 这是一个WXML示例 -->
<view class="container">
  <text>你好，微信小程序！</text>
  <button bindtap="handleClick">点击我</button>
</view>
```

#### 第二课：理解WXSS（类似CSS）
```css
/* 这是一个WXSS示例 */
.container {
  padding: 20rpx;
  background-color: #f5f5f5;
}

button {
  background-color: #07c160;
  color: white;
}
```

#### 第三课：理解JavaScript逻辑
```javascript
// 这是一个JS示例
Page({
  data: {
    message: 'Hello World'
  },
  
  handleClick() {
    console.log('按钮被点击了！')
    this.setData({
      message: '你点击了按钮'
    })
  }
})
```

## 第五步：动手修改

### 练习1：修改首页标题
1. 打开 `pages/index/index.wxml`
2. 找到 `<view class="title">微信提醒工具</view>`
3. 修改为 `<view class="title">我的提醒助手</view>`
4. 保存文件，观察右侧预览变化

### 练习2：修改按钮颜色
1. 打开 `app.wxss`
2. 找到 `.button` 样式
3. 修改 `background-color: #07c160;` 为 `background-color: #ff6b6b;`
4. 保存文件，观察按钮颜色变化

### 练习3：添加新的数据
1. 打开 `pages/index/index.js`
2. 在 `data` 中添加一个字段：
   ```javascript
   data: {
     todayCount: 0,
     todayReminders: [],
     userName: 'ZZ'  // 新增
   }
   ```
3. 在 `index.wxml` 中使用：
   ```html
   <view>欢迎，{{userName}}！</view>
   ```

## 常见问题解决

### Q1：保存文件后没有变化？
- 检查是否保存成功
- 尝试点击"编译"按钮
- 检查控制台是否有错误

### Q2：预览显示空白？
- 检查控制台错误信息
- 检查文件路径是否正确
- 尝试重启开发者工具

### Q3：代码写错了怎么办？
- 使用"撤销"快捷键（Ctrl+Z / Cmd+Z）
- 参考我提供的示例代码
- 搜索错误信息寻求帮助

## 学习建议
