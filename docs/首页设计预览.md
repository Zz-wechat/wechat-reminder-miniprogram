## 🎨 颜色方案

### 主要颜色：
- **绿色按钮**：#07c160 (微信绿)
- **蓝色按钮**：#007aff (苹果蓝)
- **卡片背景**：#ffffff (白色)
- **页面背景**：#f5f5f5 (浅灰)

### 文字颜色：
- **标题**：#333333 (深灰)
- **正文**：#666666 (中灰)
- **辅助文字**：#999999 (浅灰)

## 📱 响应式设计

### 手机尺寸适配：
- **iPhone SE**：375×667 像素
- **iPhone 12**：390×844 像素
- **常见安卓**：360×640 像素

### 布局特点：
- 所有元素使用 `rpx` 单位（响应式像素）
- 卡片有圆角和阴影效果
- 按钮有按压反馈效果
- 列表项有动画效果

## 🔄 交互流程

### 用户操作路径：
```
打开小程序 → 首页 → 点击"创建新提醒" → 填写提醒信息 → 保存
              ↓
        点击"查看所有提醒" → 管理提醒列表 → 删除/编辑
```

### 按钮功能：
1. **＋ 创建新提醒**：跳转到创建页面
2. **📋 查看所有提醒**：跳转到列表页面
3. **提醒项点击**：查看提醒详情（后续添加）

## 📊 数据展示

### 首页显示：
- 今日提醒数量：动态计算
- 提醒列表：按时间排序
- 空状态提示：没有提醒时的友好提示

### 状态标识：
- ⏰ 待提醒：绿色边框
- ✅ 已完成：灰色显示
- 🔔 已通知：特殊标记
