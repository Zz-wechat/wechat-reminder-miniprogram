# 微信小程序开发环境搭建指南

## 📦 必装软件清单

### 1. 微信开发者工具
- **作用**：小程序的开发、调试、预览、发布工具
- **下载**：https://developers.weixin.qq.com/miniprogram/dev/devtools/download.html
- **安装步骤**：
  1. 下载适合你系统的版本
  2. 双击安装包安装
  3. 打开软件，用微信扫码登录

### 2. Visual Studio Code（推荐）
- **作用**：代码编辑器，写代码更方便
- **下载**：https://code.visualstudio.com/
- **推荐插件**：
  - `minapp`：小程序语法高亮
  - `WeChat Snippet`：代码片段
  - `Live Server`：本地服务器

### 3. Node.js（可选但推荐）
- **作用**：JavaScript运行环境，后续可能用到
- **下载**：https://nodejs.org/
- **版本**：选择LTS（长期支持版）

## 🏃‍♂️ 快速开始步骤

### 步骤1：安装微信开发者工具
```
1. 访问下载页面
2. 下载对应系统版本
3. 安装并打开
4. 微信扫码登录
```

### 步骤2：创建第一个小程序
```
1. 打开微信开发者工具
2. 点击"新建项目"
3. 选择"小程序"
4. 项目设置：
   - 项目名称：MyReminder
   - 目录：选择一个空文件夹
   - AppID：可以使用"测试号"
   - 模板：选择"空白模板"
5. 点击"新建"
```

### 步骤3：了解项目结构
创建成功后，你会看到：
```
MyReminder/
├── pages/          # 页面文件夹
├── utils/          # 工具函数
├── app.js         # 小程序入口文件
├── app.json       # 全局配置
├── app.wxss       # 全局样式
└── project.config.json # 项目配置
```

## 📚 基础概念速记卡

### 小程序四大文件类型：
1. **.json** - 配置文件（数据）
2. **.wxml** - 模板文件（结构，类似HTML）
3. **.wxss** - 样式文件（样式，类似CSS）
4. **.js** - 逻辑文件（行为，JavaScript）

### 核心文件说明：
- `app.js` - 小程序的入口文件
- `app.json` - 小程序的全局配置（页面路径、窗口样式等）
- `app.wxss` - 小程序的全局样式

## ⚠️ 常见问题处理

### 问题1：安装失败
- **检查**：系统版本是否支持
- **解决**：以管理员身份运行安装包

### 问题2：扫码登录失败
- **检查**：微信是否是最新版本
- **解决**：重新扫码或重启开发者工具

### 问题3：创建项目报错
- **检查**：目录路径是否正确，是否有中文或特殊字符
- **解决**：使用纯英文路径
