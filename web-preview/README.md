# 微信提醒工具 - 静态预览版

## 📦 文件说明

这个HTML文件包含了微信提醒工具小程序的完整界面预览，具有以下特点：

### ✅ 功能特性：
1. **完整界面模拟**：1:1还原小程序设计
2. **交互式操作**：所有按钮都可点击
3. **动态数据**：可添加/删除示例提醒
4. **主题切换**：支持深色/浅色模式
5. **响应式设计**：适配各种屏幕尺寸

### 🎯 使用方式：
1. 将本文件夹下载到公司电脑
2. 双击 `index.html` 文件
3. 在浏览器中打开（推荐Chrome/Edge）
4. 不需要任何网络连接

### 🔧 演示功能：
- **点击"＋ 创建新提醒"**：显示创建流程说明
- **点击"📋 查看所有提醒"**：显示列表功能说明
- **点击"添加示例提醒"**：添加模拟提醒数据
- **点击"清空提醒"**：清空所有提醒
- **点击"切换主题"**：切换深色/浅色模式

## 💡 与真实小程序的对应关系

### 文件对应：
```
HTML文件          →   微信小程序文件
index.html       →   pages/index/index.wxml + .wxss
styles (CSS)     →   .wxss 样式文件
scripts (JS)     →   .js 逻辑文件
```

### 功能对应：
- HTML按钮点击 → 小程序按钮 bindtap
- JavaScript数据更新 → 小程序 setData
- CSS样式 → WXSS样式（略有不同）
